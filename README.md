1️⃣ Network Traffic Analysis using Wireshark
📖 Overview

This project focuses on analyzing real-world network traffic using Wireshark and Zeek to detect suspicious activities, anomalies, and potential cyber attacks. The analysis was performed on multiple PCAP files simulating real attack scenarios.

🎯 Objectives
Monitor and analyze network packets
Detect anomalies like malware, DNS tunneling, and C2 communication
Identify suspicious protocols and behaviors
Gain hands-on experience in packet-level investigation

2️⃣ VoIP Flood Attack Detection using Machine Learning
📖 Overview

This project presents a Machine Learning-based Intrusion Detection System for detecting VoIP flooding attacks such as RTP Flood and REGISTER Flood using advanced models and ensemble techniques.

🎯 Objectives
Detect VoIP-based DoS attacks
Improve detection accuracy using ML models
Identify unknown attacks using anomaly detection
